## 프로젝트 소개

JAVA 프로그래밍 수업의 프로젝트 결과물입니다. 다양한 기믹이 있는 벽돌깨기 게임을 개발하였습니다. 게임을 보다 재미있게 만들기 위해, 6개의 특수 블록과 저장된 공을 발사하는 기능을 추가하였습니다.
